# ZAI-CTS Verification Evidence Test Packages

These ZIP packages are local test fixtures for the Verification Case Dashboard.

Each ZIP contains:
- 00_manifest.json with project_code, monitoring period, district, verifier and category list
- Boundary document
- Monitoring report
- Carbon calculation
- Biomass inventory
- Satellite imagery metadata
- Field photo placeholder
- Inspection form placeholder
- Drone imagery placeholder
- Verifier statement
- Accreditation certificate
- Digital signature

Use the Verification tab, start/select a verification case, then drag one ZIP into the Evidence Package drop zone and upload it.

Important: these are workflow test fixtures, not real MRV evidence. They are intentionally small placeholder files so the upload, categorization, hashing, checklist, audit timeline and validation UI can be tested quickly.
